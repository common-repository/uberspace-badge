== Description ==

Adds a Uberspace badge to you're sidebar. You can choose between white and black.

== Installation ==

1. Upload the plugin folder to the '/wp-content/plugins/' directory.
2. Activate it in the WordPress plugin menu.
3. Move the widget to the chosen sidebar.
